# Lodestone Achievements Mini

ロードストーンのキャラクターページに、アチーブメント確認サイト「Lalachievements」へのリンクボタンを追加する軽量な Chrome 拡張機能です。

## 機能

- **Achievements ボタンの追加**: キャラクター名の下に、アチーブメントページへ直行できるボタンを追加します。
- **設定不要**: インストールするだけで動作します。複雑な設定画面や API キーは必要ありません。
- **軽量動作**: 必要なリソースのみを読み込むため、ブラウザへの負荷が最小限です。

## インストール方法

1. このフォルダ全体をダウンロードまたはクローンします。
2. Chrome で `chrome://extensions/` を開きます。
3. 右上の「デベロッパーモード」を ON にします。
4. 「パッケージ化されていない拡張機能を読み込む」をクリックし、このフォルダを選択します。

## 使い方

1. ロードストーンのキャラクターページを開きます。
2. キャラクター名の下に「Achievements」ボタンが表示されます。
3. クリックすると別タブでアチーブメント達成状況が開きます。
