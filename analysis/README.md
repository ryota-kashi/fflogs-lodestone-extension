# xivanalysis Button for FFLogs

FFLogs のレポートページを解析するために、xivanalysis への直接リンクボタンを追加する Chrome 拡張機能です。

## 主な機能

- **ダイレクト連携**:
  - FFLogs のレポート表示中に、「xivanalysis」ボタンをクリックするだけで、その戦闘ログの分析ページを別タブで開きます。
- **動的な更新**:
  - Fight（戦闘項目）を切り替えると、自動的にリンク先も更新されます。

## インストール方法

1. この `analysis` フォルダをダウンロードまたはクローンします。
2. Chrome で `chrome://extensions/` を開きます。
3. 右上の「デベロッパーモード」を ON にします。
4. 「パッケージ化されていない拡張機能を読み込む」をクリックし、この `analysis` フォルダを選択します。

## 使い方

1. FFLogs で任意のレポートページを開きます。
2. 戦闘選択エリアの横にオレンジ色の「xivanalysis」ボタンが表示されます。
3. クリックすると、現在選択されている戦闘の分析結果が表示されます。
