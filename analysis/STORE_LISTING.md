# Chrome Web Store 掲載用テキスト案

Chromeウェブストアのダッシュボードで設定できる「説明文」の多言語対応案です。

---

## 🇯🇵 日本語 (Japanese)

### 概要 (Short Description)

FFLogsのレポートページにxivanalysisへの直接リンクボタンを追加します。

### 詳細説明 (Detailed Description)

FFLogsの戦闘ログを素早く分析したいと思ったことはありませんか？
この拡張機能は、FFLogsのレポートページに「xivanalysis」への直接リンクボタンを追加し、1クリックで詳細な分析結果へアクセスできるようにします。

**主な特徴：**

- **ワンクリック連携**: レポート画面から直接xivanalysisの分析ページを開けます。
- **個人分析対応**: キャラクターごとのログ（source）を表示している際は、自動的に個人用の「Personal」ボタンも表示されます。
- **動的なリンク更新**: 戦闘項目（Fight）を切り替えると、リンク先URLもリアルタイムに更新されます。
- **シンプルなデザイン**: FFLogsのUIに馴染むよう設計されています。

## 🇺🇸 英語 (English)

### Summary (Short Description)

Adds direct links to xivanalysis on FFLogs report pages.

### Detailed Description (Detailed Description)

Ever wanted to jump straight from FFLogs to your analysis?
This extension adds a convenient "xivanalysis" button to the FFLogs report page, allowing you to access detailed combat analysis with a single click.

**Key Features:**

- **Seamless Integration**: Open the xivanalysis page directly from your FFLogs report.
- **Personal Analysis Support**: Automatically shows a "Personal" analysis button when viewing individual character logs.
- **Dynamic Link Updates**: Switching the selected fight on FFLogs will instantly update the analysis link.
- **Clean Design**: Designed to fit naturally into the FFLogs UI.
