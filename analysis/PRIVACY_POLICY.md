# Privacy Policy / プライバシーポリシー

**Last Updated / 最終更新日**: 2026-02-19

---

## English

### Overview

"xivanalysis for FFLogs" is a Chrome extension that adds a convenience button to FFLogs report pages, linking directly to the xivanalysis website.

### Data Collection

This extension **does not collect, store, transmit, or share any personal data or user information**.

### What This Extension Does

- Reads the current page URL on `fflogs.com/reports/*` to extract the report ID and fight ID.
- Injects a button element into the page linking to `xivanalysis.com`.

### What This Extension Does NOT Do

- Does not collect browsing history or personal data.
- Does not use cookies or local storage.
- Does not send any data to external servers.
- Does not use analytics or tracking of any kind.
- Does not use remote code.

### Permissions

- **Host Permissions** (`fflogs.com`): Required solely to inject the link button into FFLogs report pages and read the report/fight IDs from the URL.

### Contact

If you have any questions about this privacy policy, please open an issue on the [GitHub repository](https://github.com/ryota-kashi/fflogs-lodestone-extension).

## 日本語

### 概要

「xivanalysis for FFLogs」は、FFLogsのレポートページにxivanalysisへの直接リンクボタンを追加するChrome拡張機能です。

### データの収集について

本拡張機能は、**個人データやユーザー情報の収集・保存・送信・共有を一切行いません**。

### 本拡張機能が行うこと

- `fflogs.com/reports/*` のページURLからレポートIDと戦闘IDを読み取ります。
- ページ上に `xivanalysis.com` へのリンクボタンを挿入します。

### 本拡張機能が行わないこと

- 閲覧履歴や個人データの収集は行いません。
- Cookie やローカルストレージは使用しません。
- 外部サーバーへのデータ送信は行いません。
- アナリティクスやトラッキングは一切使用しません。
- リモートコードは使用しません。

### 権限について

- **ホスト権限** (`fflogs.com`): FFLogsのレポートページにリンクボタンを挿入し、URLからレポート/戦闘IDを取得するためにのみ必要です。
