# FFLogs Lodestone Link v2.0.0

Final Fantasy XIV のロードストーンキャラクターページを拡張し、FFLogs、Tomestone.gg、Lalachievements へのリンクや戦闘ログデータを直接表示する Chrome 拡張機能です。

## 主な機能

- **マルチプラットフォーム連携**:
  - **Achievements (Lalachievements)**: キャラクターのアチーブメント達成状況を確認可能。
  - **FFLogs**: 戦闘ログや各種 parses データを素早くチェック。
  - **Tomestone.gg**: 装備の更新履歴やキャラクター情報を確認。
- **Best % 表示 (FFLogs API連携)**:
  - FFLogs API キーを設定することで、最新コンテンツの戦闘成績（Best %）をロードストーン上に直接表示します。
- **表示カスタマイズ**:
  - 必要のないボタンや情報をオプション画面から個別にオン/オフ可能です。
- **知る人ぞ知る「隠し設定」**:
  - 初期状態はアチーブボタンのみのシンプルな構成ですが、オプション画面で特定の操作を行うことで高度な設定を開放できます。

## インストール方法

1. このフォルダ全体をダウンロードまたはクローンします。
2. Chrome で `chrome://extensions/` を開きます。
3. 右上の「デベロッパーモード」を ON にします。
4. 「パッケージ化されていない拡張機能を読み込む」をクリックし、このフォルダを選択します。

## 使い方

1. ロードストーンのキャラクターページを開きます。
2. キャラクター名の下に連携ボタンが追加されます（デフォルトでは Achievements のみ）。
3. ボタンをクリックすると、各外部サイトの該当キャラクターページが別タブで開きます。

### 隠し設定（FFLogs / Tomestone）の有効化

より高度な機能を使いたい場合は、以下の手順で設定を行ってください：

1. ブラウザの拡張機能一覧から「FFLogs Lodestone Link」の「オプション」を開きます。
2. 画面右下のバージョン番号 **「v2.0.0」を 1 回クリック** します。
3. 隠されていた FFLogs 用の API キー入力欄や、各ボタンの表示スイッチが現れます。
4. 設定を変更して「保存」をクリックし、ロードストーンのページをリロードしてください。

## ファイル構成

```

├── manifest.json      # 拡張機能設定
├── content.js         # ページ拡張メインロジック
├── options.html/js    # 設定画面
├── styles.css         # プレミアムUIスタイル
├── icon*.png          # 各種アイコン
└── README.md          # このファイル
```

## 技術仕様

- **Manifest Version**: 3
- **権限**: `storage`（設定保持用）
- **対応ドメイン**: FF14公式ロードストーン (jp/na/eu/de/fr など全地域)
